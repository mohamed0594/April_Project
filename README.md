Plateforme de Prévention Santé
 Description

Ce projet est un site web éducatif dédié à la **sensibilisation des jeunes et des adultes** aux dangers des substances addictives, notamment :

 Le tabac
 L’alcool
 Les drogues

L’objectif principal est de **prévenir les comportements à risque** et de promouvoir un mode de vie sain et responsable.

---

 Objectifs du projet
Sensibiliser sur les dangers des addictions
Informer avec des données fiables
Encourager la prévention et le changement de comportement
 Proposer des conseils santé pratiques

 Structure du site

Le site est composé de plusieurs pages :

 Effets, dangers et statistiques
 Impacts physiques et mentaux
 Recommandations, suivi et témoignages


Technologies utilisées
 HTML5
 CSS3 (Flexbox, animations, responsive design)
 Aucune utilisation de JavaScript


 Responsive Design

Le site est entièrement 

 Adapté aux téléphones 
 Compatible tablettes 
 Optimisé pour ordinateurs 



Sources des données

Les informations présentées proviennent d’organisations fiables :

 Organisation Mondiale de la Santé (OMS)
 CDC (Centers for Disease Control and Prevention)
 UNODC (Office des Nations Unies contre la drogue et le crime)
